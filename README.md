# 504班数学世界 🎨📐

> 一个以「几何美术馆」为设计语言的班级数学评价系统

## 设计理念

用三角形、圆形、正方形三种基础几何形状，构建一个温暖、专业、有504班特色的数学学习空间。

```
○ 圆形 = 珊瑚橙 = 温暖与活力
△ 三角形 = 天空蓝 = 探索与成长  
□ 正方形 = 薄荷绿 = 稳定与进步
```

## 功能模块

- ⭐ 星级管理 — 几何积分系统
- 🏆 战队排行 — 几何战队标识
- 🎮 24点游戏 — 几何迷宫挑战
- 📊 数据看板 — 几何图形可视化
- 🎁 成就系统 — 几何徽章收集

## 技术栈

- React 19 + TypeScript
- Tailwind CSS
- Express + SQLite
- Socket.io（实时通信）

## 本地运行

```bash
npm install
cp .env.example .env
# 配置 GEMINI_API_KEY（可选）
npm run dev
```
